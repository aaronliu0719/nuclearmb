# 核能流言終結者官方網站
歡迎提供建議或fork。

## 如何運行
1. 請先安裝Sass編譯器及Jekyll。

2. 執行，
	
	   make # runs at http://localhost:4000
	
	（包含Sass及Jekyll實時編譯及伺服器運行）
